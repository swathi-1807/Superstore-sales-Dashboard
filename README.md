 Superstore Sales Dashboard

 Project Overview
Analyzed a retail Superstore dataset with 9000+ records using 
Microsoft Excel to identify key business insights and built 
an interactive sales dashboard.

 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning

 Dashboard Preview
![Dashboard](dashboard_screenshot.png)

 Key Insights
- Technology category has the highest sales and profit
- West region contributes the most to overall sales
- Sales grew consistently from 2015 to 2018
- Furniture category has the lowest profit margin

 Dataset
- Source: Kaggle — Superstore Sales Dataset
- Records: 9,000+ rows
- Columns: Order ID, Category, Region, Sales, Profit, Date
